# 🕵️ Advanced SSH Honeypot & Attack Analysis System

A custom-built SSH honeypot designed to simulate a vulnerable SSH server and capture attacker behavior such as login attempts, credentials, and commands.

This project helps in understanding real-world attack patterns in a controlled environment.

---

## 🚀 Features

- 🖥 Fake SSH service (runs on custom port)
- 🔑 Credential capture (username & password logging)
- 💻 Command execution simulation
- 📡 Attacker IP logging
- 🚨 Brute-force attack detection
- 📂 Structured logging system
- 📊 Attack pattern analysis

---

## 🛠 Tech Stack

- Python 3
- Paramiko (SSH protocol)
- Linux / Kali Linux
